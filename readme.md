# NANA
這是一個用laravel使用vue建制SPA的簡單專案，參考網址 https://nana-eyelash.com/

#使用套件
* vue-router
* vue-modal
* portal-vue
* Flickity slider
* instagram api
* Tailwind css
* aos
* in-viewport

#使用方式
```bash
git clone https://github.com/Mayvis/nana.git
cd nana && composer install && npm install
npm run dev
```
